1. docker环境下，创建portainer_data文件夹
```
docker volume create portainer_data
```

1. 安装：
	1. 企业版：

```
docker run -d -p 8000:8000 -p 9443:9443 --name portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ee:lts
```
2. 社区版：

```
docker run -d -p 8000:8000 -p 9443:9443 --name portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ce:lts
```

|   |   |
|---|---|
|参数|选项|
|-d|后台运行容器（Detached mode）|
|-p 8000:8000|端口映射：宿主机端口 （此处指你mac的端口）8000 → 容器端口 8000|
|-p 9443:9443|端口映射：宿主机端口 （此处指你mac的端口） 9443 → 容器端口 9443|
|--name portainer|指定容器名称为 portainer（可以修改为你想要的名称）|
|--restart=always|自动重启策略（就是如果portainer服务停了，会自动重启）|
|-v /var/run/docker.sock:/var/run/docker.sock|挂载 Docker 套接字文件到容器（不要改动）|
|-v portainer_data:/data|挂载数据卷 portainer_data 到容器的 /data 目录（配置文件目录映射到上面我们创建的目录）|
|portainer/portainer-ee:lts|指定镜像名称与标签（企业版）|
|portainer/portainer-ce:lts|指定镜像名称与标签（社区版）|

#### 重启 Docker 生效  
```
    orb restart docker
```
